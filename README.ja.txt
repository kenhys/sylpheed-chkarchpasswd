
添付ファイル パスワードチェック プラグイン
================================================================================

このプラグインはメール送信時に添付ファイルとしてzipアーカイブが含まれている場合
パスワードが設定されていることを確認するためのプラグインです。

パスワードが設定されていないアーカイブをうっかり送信してしまうことを
気をつけることができるかも知れません。

動作環境
--------------------------------------------------------------------------------

Windows XP SP3
Windows 7

インストール方法
--------------------------------------------------------------------------------

1. dllを配置

chkarchpasswd.dllをSylpheedのプラグインディレクトリへとコピーします。

インストーラで導入している場合には%APPDATA%/Sylpheed/plugins以下に配置します。

Sylpheedのzipアーカイブを利用している場合はpluginsディレクトリへ配置するだけです。

2. 7-zip32.dllの配置

Common Archivers Libraryより7-zip32.dllを入手しsylpheed.exeと同じところに配置してください。
7-zip32.dllは以下のURLより入手することができます。

http://www.csdinc.co.jp/archiver/index.html

これでセットアップは完了です。
問題なければ鍵アイコン(赤いマイナスなし)が表示されるはずです。

使い方
--------------------------------------------------------------------------------

[ツール]-[パスワードチェック設定]より設定を行うことができます。

- 起動時にプラグインを有効化
  起動時にプラグインの機能を有効化します。

- パスワードがない場合、送信時再確認を行う
  添付ファイルにパスワードがない場合に、送信するか二度確認します。

- 添付ファイルのパスワードを検証する
  添付ファイルにパスワードがかかっている場合、送信した時点でアーカイブごとに
  パスワードを確認します。パスワードが一致しない場合送信をとりやめます。
  
注意事項
--------------------------------------------------------------------------------

あまりサイズの大きいファイルを添付した状態だと問題が出るかも知れません
