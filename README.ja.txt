
添付ファイル パスワードチェック プラグイン
================================================================================

このプラグインはメール送信時に添付ファイルがzipアーカイブならば
パスワードが設定されていることを確認するためのプラグインです。

動作環境
--------------------------------------------------------------------------------

Windows XP SP3


インストール方法
--------------------------------------------------------------------------------

1. dllを配置

chkarchpasswd.dllをSylpheedのプラグインディレクトリへとコピーします。
Sylpheedのzipアーカイブならpluginsディレクトリへ配置するだけです。

もしくは%APPDATA%/Sylpheed/plugins以下に配置します。

2. 7-zip32.dllの配置

Common Archivers Libraryより7-zip32.dllを入手しsylpheed.exeと同じところに配置してください。

http://www.csdinc.co.jp/archiver/index.html

これでセットアップは完了です。
問題なければ鍵アイコン(赤いマイナスなし)が表示されるはずです。

使いかた
--------------------------------------------------------------------------------

本プラグインは常時有効なため特に操作は必要ありません。

注意事項
--------------------------------------------------------------------------------

プラグインを無効化する場合、dllをpluginディレクトリから除去してください。
